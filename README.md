## Execução

<h4>1. Clonagem</h4>
```bash
git clone https://github.com/marllonmendez/stories.git
```

<h4>2. Instalação das dependências</h4>
```bash
npm install
```

<h4>3. Build do projeto</h4>
```bash
npm run build
```

<h4>4. Execução local</h4>
```bash
npm run dev
```
